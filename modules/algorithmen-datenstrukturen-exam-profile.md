# Prüfungsprofil: Algorithmen & Datenstrukturen
Analysiert: 5 Klausuren (2020 – 2024)

## Themen nach Häufigkeit

| Thema | Auftreten | Aufgabentyp | Prüfungsrelevanz |
|-------|-----------|-------------|-----------------|
| O-Notation / Laufzeitanalyse | 5/5 | Berechnung, Beweis | 🔴 hoch |
| Dijkstra / kürzeste Wege | 4/5 | Berechnung, Trace | 🔴 hoch |
| Dynamic Programming | 4/5 | Berechnung, Erklärung | 🔴 hoch |
| Heap / Priority Queue | 3/5 | Implementierung, Trace | 🟡 mittel |
| Rekursion / Mastertheorem | 3/5 | Beweis, Berechnung | 🟡 mittel |
| Graphtraversierung (BFS/DFS) | 2/5 | Trace | 🟡 mittel |
| AVL-Bäume | 1/5 | Trace | 🟢 niedrig |

## Typische Aufgabenformulierungen

- "Geben Sie die Laufzeit des folgenden Algorithmus in O-Notation an und begründen Sie."
- "Führen Sie Dijkstra auf dem folgenden Graphen aus und geben Sie die Distanztabelle an."
- "Entwerfen Sie einen DP-Algorithmus für folgendes Problem. Geben Sie Rekurrenz und Laufzeit an."
- "Beweisen Sie per Induktion, dass..."

## Nie gefragt (trotz Vorlesungsinhalt)

- Fibonacci-Heap (Implementierungsdetails)
- Amortisierte Analyse
- Externe Sortierverfahren
